# Futuro-do-trabalho-IA
Este projeto foi desenvolvido para o desafio “Era das IAs Generativas” da DIO. Ele representa uma visão artística e inspiradora do futuro do trabalho, em que a colaboração entre humanos e inteligências artificiais redefine produtividade, criatividade e propósito profissional.  A imagem gerada por IA retrata um ambiente de trabalho futurista.
